Introduction to JavaScript for…of Loop in ES6

Summary: in this tutorial, we will introduce you to the JavaScript for...of statement that allows you to create a loop iterating over an iterable object.

Introduction to the JavaScript for...of loop

ES6 introduced a new construct for...of that creates a loop iterating over an iterable object such as an Array, a Map, a Set, or any object that implements the iterator protocol .

The following illustrates the syntax of the for...of syntax:

for (variable of iterable) {
   // statements 
}

variable

The variable is any property of an object.

object

An iterable object such as an Array, a Set, a Map, etc.

JavaScript for of loop examples

Let’s take a look at some examples of using the for...of loop.

Iterating over an array

The following example shows you how to use the for...of to iterate over elements of an array.

let scores = [80, 90, 70];
for (let score of scores) {
    score = score + 5;
    console.log(score);
}

If you don’t change variable inside the loop, you should use the  const instead of let as follows:

for (const score of scores) {
    console.log(score);
}
// 80
// 90
// 70

Iterating over a string

The following example uses the for...of loop to iterate over characters of a string.

let str = 'abc';
for (let c of str) {
    console.log(c);
}
// a
// b
// c

Iterating over a map

The following example illustrates how to use the for...of statement on a Map object.

var colors = new Map();
colors.set('red', '#ff0000');
colors.set('green', '#00ff00');
colors.set('blue', '#0000ff');

for (let color of colors) {
    console.log(color);
}

// ["red", "#ff0000"]
// ["green", "#00ff00"]
//["blue", "#0000ff"]

Iterating over a set

The following example shows how to iterate over a set using the for...of loop

let nums = new Set([1, 2, 3]);

for (let num of nums) {
    console.log(num); //
}
// 1
// 2
// 3

for...of vs. for...in

The for...in loop iterates over all enumerable properties of an object. It does not iterate over a collection such as Array, Map or Set.

Unlike the for...in loop, the for...of loop iterates a collection, rather than an object. In fact, the for...of iterates over elements of any collection that has the [Symbol.iterator] property.

The following example illustrates the differences between for...of and for...in

let numbers = [6, 7, 8];
numbers.foo = "foo";

for (let i in numbers) {
    console.log(i);
}
// 0
// 1
// 2
// foo
for (let i of numbers) {
    console.log(i);
}
// 6
// 7
// 8

In this tutorial, you have learned how to use the JavaScript for...of loop to iterates over a collection.
